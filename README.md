# TinyURL.com
Project for shorten links (url) and then share them. And converting Short URL back to Original URL

## Converting URL to Short URL
POST Request : http://localhost:8081/
Body:
  {
  "url":"https://www.youtube.com/shorts/zaMzhgjgcJQ"
  }
